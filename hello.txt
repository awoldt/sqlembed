To split a string and loop through the parts in Rust, use the split() method directly in a for loop. The split() method returns an iterator of string slices (&str), so you do not need to allocate memory or collect the items into a vector first.By a Custom Character DelimiterUse this approach to split a string by commas, colons, or any specific character:rustfn main() {
    let text = "apple,banana,cherry";

    // Loop directly over the split iterator
    for fruit in text.split(',') {
        println!("{}", fruit);
    }
}
Use code with caution.By Whitespace (Words)If you want to split a sentence by spaces or tabs, use the split_whitespace() method. This automatically handles multiple consecutive spaces and trailing whitespace:rustfn main() {
    let sentence = "Rust  is    awesome";

    for word in sentence.split_whitespace() {
        println!("{}", word);
    }
}
Use code with caution.By LinesTo split a multi-line string line-by-line, use the lines() method:rustfn main() {
    let paragraph = "Line 1\nLine 2\nLine 3";

    for line in paragraph.lines() {
        println!("{}", line);
    }
}
Use code with caution.Pro-Tip: Memory OwnershipThe variables inside your loop (like fruit or word) are string slices (&str). They are borrowed references pointing back to your original string.
